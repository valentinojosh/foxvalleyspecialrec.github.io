Fox Valley Special Recreation Association
Incident Management System

The FVSRA IMS is a web application created to replace the current paper report system for the local business, Fox Valley Special Recreation Association. The system is meant to allow employees and admins to create forms, have them update into a database, and when requested have the information output to an Excel file. The demo video is currently a demo of the prototype. It will be updated demoing the fully functional deployed application no later than May 10th.
