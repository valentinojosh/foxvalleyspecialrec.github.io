Fox Valley Special Recration Assocation
Incident Management System

The FVRSA IMS is a web application created to replace the current paper report system for the local buisness, Fox Valley Special Recreation Association. This project was a part of my Software Engineering course at Aurora University. The goal of the project was to create a prototype using HTML, CSS, JavaScript, Node.js, Express, and SQL. It was developed in an agile sprint environment with a team of four people. The system is meant to allow employees and admins to create forms, have them update into a database, and when requested have the information output to an Excel file. Due to the nature of the course, the prototype was not meant to be fully functional. Currently, the system is semi-functional. For further information, please scroll down to the BLANK section and select the explanation word file. This file was compiled by the entire team, however, I have labeled the sections that I was responsible for. Overall I put in the most hours of the team, and I am proud of the progress we made. In the explanation doc you will see our sprints, in which our hours committed to the project were logged. These hours only included the log of work towards the prototype itself and not towards documentation. lower buttons: -desc word doc -link to the original forms we were given to copy
